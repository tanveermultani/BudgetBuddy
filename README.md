# BudgetBuddy
**BudgetBuddy** is a personal finance tracker designed to help you manage your income, expenses, and budget goals. It provides a simple yet powerful interface for tracking transactions, categorizing expenses, and visualizing your spending patterns over time. This project is built with the aim of improving personal financial awareness and assisting users in making informed financial decisions.

## **Features**
- **Add and Manage Transactions:** Track income and expenses with details like amount, date, and category.
- **Budget Tracking:** Set monthly or weekly budgets to monitor spending against your goals.
- **Transaction History:** View a detailed list of all transactions with filters by category and date range.
- **Data Visualization:** Visualize spending patterns through charts, helping identify trends and areas to save.
- **Customizable Categories:** Define custom categories to tailor the tracker to your specific needs.
